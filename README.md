# Bubble Sort Visualizer
This program provides a clear and understandable visual representation of the Bubble Sort algorithm utilizing the Python Pygame Library. Users can determine the number of elements to be sorted, as well as the fps in which they are sorted. 

## Instructions

1. Clone Repository:
  git clone https://github.com/aaronnewmanj/Bubble-Sort-Visualizer.git

2. Run Program:
   python3
